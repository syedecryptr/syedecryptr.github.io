Some Work by me:

Karnaugh Map Solver 

A web application using Javascript which simplifies boolean algebra expression containing upto 8 variables, keeping same logic and reducing number of gates that will be used for implementing same logic, using Quine - McCluskey Algorithm and Petric's method .

2048 Game

A fully functional famous 2048 game adding advance features like dynamic number of rows and columns, sending score to your email address, in python using PyGame. Similar game is written in C language including features like file handling, sorting playes according to their score in file by itself :- 2048

Autonomous RF Car

A python implementation of neural network algortihm of machine learning, training a RF car with attached camera to drive by itself. Images taken by car were sent over wifi from car to processor where neural networks trains model to predict output based on input image. Further based on those outputs recieved from trained model using Py-serial RF car was controlled

Arduino Mega-Cricket

Cricket Game made using Arduino an open-source electronics platform, using Liquid crystal Library to interface LCD screen with arduino and switch controllers for controlling game. This game was displayed in Nimble IIT_J technical event.

FF Transform Radix Method

Implementing DFT,Radix-4,Radix-2 and Split-Radix FFT Algorithms on a Web-App. Calculating the runtime complexity of all above algorithms and suggesting user the Best (fastest) Algorithm for his application. Favourable platforms for the implementation of the project include HTML, Bootstrap, Javascript, Plotly. 

Load-Flow Study

 Used MATLAB Power System Toolbox software to simulate and study the effects of generator rescheduling, series and shunt compensation and system load on IEEE14 bus power system Solution of Load Flow Problem. Compared the convergence and running time of various algorithms used to solve Load flow problems.
